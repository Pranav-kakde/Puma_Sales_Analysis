# Puma_Sales_Analysis


## Project Overview

This project is an interactive sales analytics dashboard developed using Google Looker Studio for analyzing Puma shoe sales data. The dashboard helps visualize key business metrics, identify trends, and generate insights for better decision-making.

---


## Objective

The goal of this project is to provide a centralized dashboard for monitoring sales performance across different locations, categories, and customer segments.

---

## Key Performance Indicators (KPIs)

The dashboard tracks the following business metrics:

| KPI | Value |
|------|--------|
| Sales Amount | 113.3M |
| Total Cost | 71.2M |
| Quantity Sold | 24.0K |
| Transactions | 7.8K |
| Maximum Sales | 7.8K |
| Profit | 42.1M |

---

## Dashboard Features

### KPI Scorecards
Provides quick business performance indicators:

- Total Sales Amount
- Total Cost
- Quantity Sold
- Transactions Count
- Maximum Sales
- Total Profit

---

### Interactive Filters

Users can dynamically filter dashboard data using:

- Product Category
- State
- Customer Name
- Order Number
- City

---

### Geographical Analysis

A map visualization displays:

- Sales distribution across states
- Regional sales performance
- Category-wise sales concentration

Insights:
- High sales concentration observed in major regions.
- Easy identification of top-performing locations.

---

### Product Category Analysis

Pie chart representation showing sales distribution across products:

Categories include:

- Single Monk
- Derby Shoes
- Sneakers
- Loafers
- Brogues
- Shoes
- Oxford Shoes

Insights:

- Single Monk category contributes the highest percentage.
- Oxford Shoes show comparatively lower contribution.

---

### Cost Analysis

Bar chart visualization used for:

- Comparing cost across categories
- Identifying high-cost segments
- Understanding spending distribution

---

### Sales Trend Analysis

Line chart visualizing:

- Sales amount over time
- Growth patterns
- Peak and low-performing periods

Insights:

- Sales peak observed around mid-period.
- Decline visible after peak sales period.

---

### State-wise Sales Table

Tabular representation including:

- State
- Quantity Sold
- Cost
- Sales Amount

Purpose:

- Detailed state-level performance analysis
- Easy comparison among regions
